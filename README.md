# project-hermes

A video streaming platform that features a front end video player that streams videos in multiple resolutions at high speed

# Why I'm doing this

I want to brush up on my programming skills for both front and back end, CD/CI knowledge and try something a bit different than what I do in my day job.

This will also be an opportunity to explore technologies that I'm less familiar with.

The project will eventually include modern expected features such as tracing, monitoring with dashboards, security and session management, cryptography.

# Planned Technology stack

Front end -> Next.js (React) + Tailwind CSS

Video Player -> video.js or Shaka Player

Back End API Node.js (Typescript) + Fastify

Streaming Media -> FFmpeg + AWS S3/CloudFront

Database -> PostgresSQL + Redis

Deployment -> Docker, Kubernetes, ArgoCD

Monitoring -> Prometheus

Logging -> Loki

Tracing -> Jaeger

Visualisation -> Grafana

# On AI

I will be using as little AI as possible for this project. However, when the project reaches a point that I am happy with and covers most of the technologies and features I will move on to more AI based features and even using AI assistance.
